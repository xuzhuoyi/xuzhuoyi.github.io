LDE Team

LDE 源代码正在每周更新

在几个月的努力下, LDE 0.0.7 版本成功发布了, 这是一个基于 Qt 4 的轻量级桌面环境, 并有一些专门开发的基础应用。而且 LDE 的源代码每周都在变化，每次都有很多安全隐患修复和新功能与新特性。LDE 需要大家的帮助以变得更加完美。以下是下载与安装 LDE 的方法以及一些开发者关心的事。

关于 LDE

LDE，全名为 Loongson Desktop Environment，原来是专为龙芯等 mipsel 平台 CPU 设计的，从 0.0.4 版本以后逐渐向各平台发展。现在能够支持 i386\AMD64 等多平台。LDE 项目旨在向龙芯用户提供一个低硬件门槛的桌面环境，在轻量化的前提下，尽可能实现更强大的功能。LDE 以 Qt 4 作为图形界面库，有强大的移植性，在 mipsel 以外的平台也可编译运行。从 1.0.0 版本开始，LDE 逐渐向 Qt 5 转移。LDE 提供了一些配套的可独立运行元件。LDE 使用 Openbox 为其默认视窗管理器，并且希望能够提供一个建立在可独立的套件上的轻巧而快速的桌面。

已经测试支持的Linux系统/版本:

  • Linux Mint 16/17/17.1
  • Debian stable/testing/sid
  • Gentoo
  • OpenSuse 13.1/Factory
  • Ubuntu 13.10/14.04/14.10

LDE 理论上支持所有 Linux 发行版。近期正准备 Windows 与 OS X 平台的移植。

下载

我们将稳定二进制程序放在 Github 的 Release 中，通过地址 https://github.com/xuzhuoyi/LDE/releases 来获取。 然而稳定版的版本可能比较老，构建于数周前。如果想要获取最新的特性，您可以下载源码手动编译。2015 年后我们也将提供 Stable 和 Testing 两个发行通道。 源码在 https://github.com/xuzhuoyi/LDE 您可以 Star\Fork\Watch 来帮助支持我们。
